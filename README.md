This repository contains the current build of a game project developed by a single developer
The project is functional, but not organized in a strict or production-ready structure. 
The focus has been on building features and systems quickly rather than maintaining a clean architecture throughout development.
This project was built entirely by one person handling all aspects of development, including:

Gameplay programming
Systems and mechanics
Assets and resources
UI and interface
Testing and debugging

Because of this, there was no dedicated time allocated to consistently restructure or standardize the project layout while working on features.

As a result, file organization, naming conventions, and folder structure may be inconsistent or partially unfinished.
